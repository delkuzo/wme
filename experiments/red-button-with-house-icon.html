Эксперимент: Красная кнопка с иконкой house

Демонстрация использования иконок Gravity UI в кастомных кнопках. Красная кнопка высотой 40px с иконкой дома.

Демо кнопки

HTML код кнопки:

<button class="red-button-40" onclick="alert('Кнопка нажата!')">
  <svg xmlns="http://www.w3.org/2000/svg" width="16" height="16" fill="none" viewBox="0 0 16 16">
    <path fill="currentColor" fill-rule="evenodd" d="M8.5 2.5a.75.75 0 0 0-1 0l-4.5 4.5a.75.75 0 0 0 1 1.06L8 4.06l3.5 3.94a.75.75 0 1 0 1-1.06L8.5 2.5ZM3.5 7.5a.75.75 0 0 0-1.5 0v4.5c0 .828.672 1.5 1.5 1.5h8c.828 0 1.5-.672 1.5-1.5V7.5a.75.75 0 0 0-1.5 0v4.5H3.5V7.5Z" clip-rule="evenodd"/>
  </svg>
  Главная
</button>

CSS стили кнопки:

.red-button-40 {
  display: inline-flex;
  align-items: center;
  gap: 8px;
  height: 40px;
  padding: 0 16px;
  background: #ff4444;
  color: white;
  border: none;
  border-radius: 8px;
  font-family: 'YS Text', sans-serif;
  font-size: 14px;
  font-weight: 500;
  cursor: pointer;
  transition: all 0.2s ease;
}

.red-button-40:hover {
  background: #e63939;
  transform: translateY(-1px);
  box-shadow: 0 4px 12px rgba(255, 68, 68, 0.3);
}

.red-button-40:active {
  background: #d63333;
  transform: translateY(0);
  box-shadow: 0 2px 8px rgba(255, 68, 68, 0.2);
}

.red-button-40 svg {
  width: 16px;
  height: 16px;
  flex-shrink: 0;
}

Особенности реализации

  • Высота 40px: Соответствует дизайн-системе для кнопок среднего размера
  • Красный цвет: #ff4444 с hover эффектом #e63939
  • Иконка house: Использована иконка из Gravity UI с размером 16x16px
  • Flexbox layout: Иконка и текст выровнены по центру с отступом 8px
  • Интерактивность: Hover и active состояния с анимацией
  • Типографика: Шрифт YS Text Medium 14px
