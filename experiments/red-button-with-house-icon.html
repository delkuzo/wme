Эксперимент: Красная кнопка с иконкой house

Особенности реализации

  • Кнопка из UI Kit: Использует классы btn btn-action btn-40 из дизайн-системы
  • Высота 40px: Соответствует дизайн-системе для кнопок среднего размера
  • Красный цвет: #F8604A с hover эффектом #E45844 (стандартные цвета UI Kit)
  • Иконка house: Правильная иконка из Gravity UI с размером 16x16px
  • Flexbox layout: Иконка и текст выровнены по центру с отступом 8px
  • Интерактивность: Hover и active состояния из UI Kit
  • Типографика: Шрифт YS Text Medium 14px
